# Sudoku solver

This project aims at solving a Sudoku from a picture, which might have a poor quality (blurred, not upright, small...).

## How to run the project
Activate the virtual environment
```bash
conda activate sudoku
```

Run the main file
```bash
python main2.py
```

## Sources
